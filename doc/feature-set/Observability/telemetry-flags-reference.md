# Telemetry Flags Reference

> **224 flags.** GENERATED by scripts/gen-telemetry-reference.py from telemetry-flags.jsonc — do not hand-edit.

Every gate that controls Typhon's [distributed tracing](xref:feature-observability-distributed-tracing) and [typed-event Profiler](../Profiler/README.md). All default **off** unless noted; enabling a subtree's root enables its leaves ([parent-implies-children](xref:feature-observability-telemetry-config-gating)). Set them in `typhon.telemetry.json` or via `TYPHON__…__ENABLED` environment variables.

## Copy-paste template
